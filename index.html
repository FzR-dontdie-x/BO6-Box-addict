Welcome to the Black Ops 6 Box Addict Calling Card Tracker

A project made by FzR dontdie x to track the Box Addict calling card

Citadelle de Morts has the shortest box list and all weapons are available from the start and is recommended for this challenge.

Liberty Falls

Liberty Falls is the first zombies map.

Please Note: In Directed Mode you must first build the Jet Gun and swap it for another weapon and let it despawn before it will appear in the box. The exception is by using a wonderbar gobblegum.
